UI components for working with [GraphQL](https://graphql.org/).

<p>&nbsp;<p>

## Setup

    yarn add @platform/ui.graphql

Ensure base `css` files are included in the consuming project.

``` typescript
import `@platform/ui.graphql/import.css`
```
