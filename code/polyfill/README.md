# polyfill
Standard JS polyfills.

<p>&nbsp;<p>


## Setup

```
yarn add @platform/polyfill
```

<p>&nbsp;<p>

## Usage

Import common polyfills (recommended):

```typescript
import '@platform/polyfill';
```


or import something more specific:

```typescript
import 'core-js/stable';
import 'regenerator-runtime/runtime';
```
