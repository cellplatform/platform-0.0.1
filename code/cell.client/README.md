A strongly typed client-library for operating a [`cell.http`](../cell.http) service end-point.

<p>&nbsp;</p>

## Setup

    yarn add @platform/cell.client

<p>&nbsp;</p>

## Test

Unit-tests that exercise this client-library can be found in within [`cell.router`](../cell.router).

```bash
cd ../cell.router
yarn test
```
